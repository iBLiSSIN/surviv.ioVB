# surviv.io mods by VN BPM on YouTube
Mods for surviv.io

Install Tampermonkey, click on the extensions icon, click on "Create a new user script" then copy and paste all content in [here](https://github.com/iBLiSSIN/SurvivMods/blob/main/SurvivModsBy%40VN%20BPM%20on%20YT.user.js). Press ctrl + s and open surviv.io (remember to reload the web page with ctrl + f5)
